# Firefox Security Audit – Basierend auf CIS‑ähnlichen Benchmarks

## Übersicht

Dieses Projekt zeigt einen Security‑Audit für den Mozilla Firefox Browser.  
Die Bewertung orientiert sich an Best‑Practices aus Browser‑Härtungsrichtlinien, wie sie auch in CIS Benchmarks für Webbrowser definiert sind.

## Ziele

- Bewertung sicherer Firefox Konfigurationen
- Identifikation von Abweichungen
- Risikobasierte Bewertung
- Konkrete Handlungsempfehlungen

## Scope

- Firefox Version ≥ ESR/Latest Stable
- Windows Clients (10/11)
- Konfigurationsdateien
- Policies (falls MDM vorhanden)

## Audit‑Kategorien

- Password/AutoFill Security
- Proxy / Network Settings
- HTTPS‑Only Mode
- Tracking Protection
- Add‑ons / Extensions
- Certificate/HTTPS strictness
- Telemetry / Data Collection

## Bewertungskriterien

- Konform
- Nicht konform
- Nicht anwendbar
- Manuelle Prüfung nötig

## Zielgruppe

IT‑Security Teams, Auditoren, DevSecOps
