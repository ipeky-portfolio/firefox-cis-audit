# Audit‑Scope (Firefox)

## In‑Scope

- Mozilla Firefox (alle relevanten Profile)
- Firefox Policies (Enterprise Konfigurationen)
- Config‑Dateien (`prefs.js`, `policies.json`)
- Add‑on / Extension‑Liste

## Out‑Of‑Scope

- Andere Browser
- Netzwerk‑Layer Sicherheitsgeräte
- Cloud‑Browser

## Voraussetzungen

- Administrative Zugriffsrechte
- Zugriff auf Benutzer‑Profile
